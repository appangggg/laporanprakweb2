1 Soal & Bentuk Awal NFA ε-move

Σ = {0, 1, 2}. Ubahlah NFA ε-move di bawah ini menjadi NFA tanpa ε-move yang ekuivalen.

graph LR start(( )) --> q0((q0)) q0 -->|0| q0 q0 -->|ε| q1((q1)) q1 -->|1| q1 q1 -->|ε| q2(((q2))) q2 -->|2| q2 style start fill:transparent,stroke:transparent

Langkah 1: Menentukan ε-closure

Himpunan state yang dapat dicapai tanpa membaca input.

  • ε-closure(q0) = {q0, q1, q2}
  • ε-closure(q1) = {q1, q2}
  • ε-closure(q2) = {q2}

Langkah 3: Menentukan State Akhir

State akhir semula adalah {q2}. State baru akan menjadi final jika ε-closure-nya memuat q2.

  • ε-cl(q0) = {q0, q1, q2} → Final
  • ε-cl(q1) = {q1, q2} → Final
  • ε-cl(q2) = {q2} → Final
Himpunan State Akhir Baru (F') = {q0, q1, q2}

Langkah 2: Menghitung Fungsi Transisi Baru (δ') Arahkan kursor ke tabel!

Rumus: δ'(q, a) = ε-closure(δ(ε-closure(q), a))

δ' 0 1 2
q0 {q0, q1, q2} {q1, q2} {q2}
q1 ∅ {q1, q2} {q2}
q2 ∅ ∅ {q2}
Lihat detail perhitungan matematis δ'

Untuk q0:

δ'(q0,0) = ε-cl(δ({q0,q1,q2},0))
= ε-cl({q0}) = {q0,q1,q2}

δ'(q0,1) = ε-cl(δ({q0,q1,q2},1))
= ε-cl({q1}) = {q1,q2}

δ'(q0,2) = ε-cl(δ({q0,q1,q2},2))
= ε-cl({q2}) = {q2}

Untuk q1:

δ'(q1,0) = ε-cl(δ({q1,q2},0))
= ε-cl(∅) = ∅

δ'(q1,1) = ε-cl(δ({q1,q2},1))
= ε-cl({q1}) = {q1,q2}

δ'(q1,2) = ε-cl(δ({q1,q2},2))
= ε-cl({q2}) = {q2}

Untuk q2:

δ'(q2,0) = ε-cl(δ({q2},0))
= ε-cl(∅) = ∅

δ'(q2,1) = ε-cl(δ({q2},1))
= ε-cl(∅) = ∅

δ'(q2,2) = ε-cl(δ({q2},2))
= ε-cl({q2}) = {q2}

2 Soal & Bentuk Awal NFA ε-move

Σ = {0, 1, 2}. Ubahlah NFA ε-move di bawah ini menjadi NFA tanpa ε-move yang ekuivalen.

graph LR start(( )) --> q0((q0)) q0 -->|0| q0 q0 -->|ε| q1(((q1))) q1 -->|1| q0 style start fill:transparent,stroke:transparent

Langkah 1: Menentukan ε-closure

  • ε-closure(q0) = {q0, q1}
  • ε-closure(q1) = {q1}

Langkah 3: Menentukan State Akhir

State akhir semula adalah {q1}.

  • ε-cl(q0) = {q0, q1} → Final
  • ε-cl(q1) = {q1} → Final
Himpunan State Akhir Baru (F') = {q0, q1}

Langkah 2: Menghitung Fungsi Transisi Baru (δ')

δ' 0 1 2
q0 {q0, q1} {q0, q1} ∅
q1 ∅ {q0, q1} ∅
Lihat detail perhitungan δ'

Untuk q0:

δ'(q0,0) = ε-cl(δ({q0,q1},0))
= ε-cl({q0} U ∅) = {q0,q1}

δ'(q0,1) = ε-cl(δ({q0,q1},1))
= ε-cl(∅ U {q0}) = {q0,q1}

δ'(q0,2) = ε-cl(δ({q0,q1},2))
= ε-cl(∅ U ∅) = ∅

Untuk q1:

δ'(q1,0) = ε-cl(δ({q1},0))
= ε-cl(∅) = ∅

δ'(q1,1) = ε-cl(δ({q1},1))
= ε-cl({q0}) = {q0,q1}

δ'(q1,2) = ε-cl(δ({q1},2))
= ε-cl(∅) = ∅
